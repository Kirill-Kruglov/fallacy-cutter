# Worked example

`CL0_preregistration.json` is a real preregistration artifact extracted from Ascesis for the CL0 gate. It is included as a format example for the kind of `PREREG.json` record checked by `gate_harness.prereg`: question, metrics, thresholds with rationale, ground-truth oracle, and positive/negative controls. The domain is specific to a safe-transition ledger; the reusable value is the structure, not the subject matter. A clean domain-neutral end-to-end example is intentionally left for a later pass.
